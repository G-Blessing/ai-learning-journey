# AI Engineer Roadmap (6 Months)

## PHASE 1: FOUNDATIONS (Weeks 1-8)
### Week 1-2: Python Fundamentals + Git/GitHub
- **Daily Tasks**: 1-2 hours of Python coding exercises.
- **Resources**:
  - Python for Everybody (Coursera)
  - DataCamp
  - HackerRank
- **Mini Projects**: Build a simple calculator, a to-do app.

### Week 3-4: Math for Machine Learning
- **Topics**:
  - Linear Algebra (3Blue1Brown)
  - Calculus
  - Statistics (Khan Academy)

### Week 5-6: Data Manipulation with Pandas, Matplotlib, Seaborn
- **Daily Tasks**: Work with Kaggle datasets to explore data.
- **Resources**: Documentation for Pandas, Matplotlib.

### Week 7-8: Introduction to ML Theory
- **Course**: Andrew Ng's Machine Learning Course.
- **Projects**: Build linear regression model from scratch.
